# Rebustle

A rebus puzzle party game built with React + TypeScript + Vite.

## Puzzle Credits

All puzzles are sourced from [JustFamilyFun.com](https://justfamilyfun.com/). Full credit and thanks to them for the fantastic rebus puzzle content.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
